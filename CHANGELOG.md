# 更新日志

## [0.1.0] - 2026-06-16

首个公开预览版本。

### 新增

- 透明置顶桌面浮窗。
- 右侧边缘隐藏与唤醒。
- 动态 Tab 暂存槽。
- 文本暂存、编辑与复制回系统剪贴板。
- 图片剪贴板收纳、预览与复制回系统剪贴板。
- 本地持久化保存 Tab、文本、图片路径和隐藏状态。
- 系统托盘菜单，支持显示、隐藏、打开数据目录、关于和退出。
- 全局快捷键 `Alt + Space` 唤醒或隐藏窗口。

### 已知限制

- 当前仅面向 Windows 公开预览。
- 安装包暂未代码签名，首次运行可能触发 Windows SmartScreen 提示。
- 暂未提供自动更新能力。
