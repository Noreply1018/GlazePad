# 更新日志

## [0.2.0] - 2026-06-20

### 新增

- `+` 单击新增空白文本 Tab，右键 `+` 收纳当前剪贴板为新 Tab。
- 新增配色设置，支持冰蓝、雾白、薄荷、玫瑰和石墨。
- 新增四档透明度设置：清晰、标准、轻透和极透。
- 新增开机自启动开关。

### 改进

- 新建 Tab 标题会复用已删除的最小空缺编号。
- 复制失败时显示具体错误，便于定位剪贴板或图片路径问题。
- 关于窗口版本号改为自动读取应用版本。
- 本地状态兼容旧版 `state.json`，旧数据会自动补齐默认设置。

## [0.1.0] - 2026-06-16

首个公开预览版本。

### 新增

- 透明置顶桌面浮窗。
- 右侧边缘隐藏与唤醒。
- 动态 Tab 暂存槽。
- 文本暂存、编辑与复制回系统剪贴板。
- 图片剪贴板收纳、预览与复制回系统剪贴板。
- 本地持久化保存 Tab、文本、图片路径和隐藏状态。
- 系统托盘菜单，支持显示、隐藏、打开数据目录、关于和退出。
- 全局快捷键 `Alt + Space` 唤醒或隐藏窗口。

### 已知限制

- 当前仅面向 Windows 公开预览。
- 安装包暂未代码签名，首次运行可能触发 Windows SmartScreen 提示。
- 暂未提供自动更新能力。
