# 产品与交互规范

GlazePad 是 Windows 桌面上的透明暂存槽，用来临时保存需要反复复用的文字或图片。

它不是完整剪贴板历史，也不是笔记软件。只有用户主动收纳进 Tab 的内容会被保存。

## 产品形态

- 透明置顶短浮窗，默认贴近屏幕右侧。
- 每个 Tab 是一个独立暂存槽，只保存一份文本或一张图片。
- 窗口高度固定，切换文本和图片时外框不跳动。
- 隐藏后收起到屏幕右侧，只保留一条细唤醒边。
- 主要反馈写在顶部状态文字，不使用 toast 或弹窗。

## 顶部操作

右上角四个按钮依次是：

1. 复制当前 Tab 内容到系统剪贴板。
2. 删除当前 Tab。
3. 新建或收纳 Tab。
4. 隐藏到屏幕右侧。

`+` 的规则：

- 单击 `+`：新建空白文本 Tab，并立即切换过去。
- 右键 `+`：读取当前系统剪贴板，收纳为新的文本或图片 Tab。
- 剪贴板为空或读取失败时，不创建 fallback Tab，只更新状态文字。

删除规则：

- 删除当前 Tab 后切到邻近 Tab。
- 至少保留一个 Tab。
- 新建 Tab 标题复用已删除的最小空缺编号，例如删除 `Tab 2` 后下次优先创建 `Tab 2`。

## 内容槽位

文本 Tab：

- 在当前槽位内直接编辑。
- 长文本在槽位内部滚动。
- 复制按钮只复制文本，不自动全选。

图片 Tab：

- 显示单张图片预览。
- 图片文件保存在本地应用数据目录。
- 底部状态显示图片类型和尺寸。
- 复制按钮把图片写回系统剪贴板。

## 隐藏与唤醒

- 点击隐藏按钮、按 `Alt + Space` 或显示态下按 `Esc` 可以隐藏。
- 隐藏态下，按 `Alt + Space`、点击右侧唤醒边或点击托盘图标可以唤醒。
- 动画应可取消，连续触发不能导致窗口刚出现又立刻收回。
- 系统开启减弱动态时，应减少或跳过位移动画。

## 托盘设置

托盘菜单负责不适合放进主浮窗的设置：

- 配色：冰蓝、雾白、薄荷、玫瑰。
- 透明度：清晰（72%）、标准（52%）、轻透（38%）、极透（26%）。
- 开机自启动：显示为“已开启”或“未开启”。
- 打开数据目录。
- 关于。
- 退出。

配色会影响面板、按钮、状态点、唤醒边和阴影。透明度应让背景、内容槽和控件有可感知差异，但文字可读性优先。

## 暂不支持

- 完整剪贴板历史。
- Tab 重命名。
- Tab 分类或多卡片列表。
- 富文本编辑。
- 图片编辑。
- 云同步。
