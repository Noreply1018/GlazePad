# GlazePad 产品说明

## 产品定位

GlazePad 是一个贴在桌面上的透明暂存槽，用来短时间保存会反复使用的文字或图片。

它不是剪贴板历史，也不是完整笔记软件。用户只会把需要复用的内容主动放进来，每个 Tab 都是一个独立槽位，槽位里永远只放一条内容。

## 核心目标

- 让用户把临时要复用的文字、代码或图片放进一个干净的桌面浮窗。
- 通过明确的复制按钮，把当前槽位内容快速写回系统剪贴板。
- 保持浮窗短、透明、克制，不把注意力从主工作区里抢走。
- 用简单的唤醒 / 隐藏机制，让它在需要时出现，不需要时退场。

## 目标用户

第一目标用户是开发者本人，使用场景集中在 Windows 桌面办公、开发、沟通回复和资料整理。

典型场景包括：

- 临时保存几段会反复发送的话术。
- 临时保存代码片段、命令、链接或备注。
- 临时保存一张截图、参考图或素材图。
- 在不同内容之间快速切换，避免反复复制粘贴原始来源。

## 产品形态

- Windows 桌面上的短浮窗，默认贴近屏幕右侧。
- 整体保持透明或半透明，像一块轻薄玻璃面板。
- 默认通过快捷键唤醒或隐藏。
- 隐藏时从右侧边缘收起，唤醒时从右侧边缘磁吸式滑出。
- 外框高度固定，文本和图片都适配到同一块固定槽位里，不因为内容类型改变整体高度。

## 当前界面结构

当前原型采用以下结构：

1. 顶部有品牌名、保存状态和一组图标按钮。
2. 右上角按钮依次是复制、删除当前 Tab、新增 Tab、隐藏。
3. 中间是一排 Tab，每个 Tab 代表一个独立暂存槽。
4. 每个 Tab 下方只有一个内容框，不是列表，也不是分类容器。
5. 文本槽位显示单个文本框，可直接编辑。
6. 图片槽位显示单张图片预览，按固定槽位铺满展示。
7. 底部显示槽位数量、当前内容类型和桌面模式状态。

## 交互规则

### 新增 Tab

点击 `+` 会立刻新增一个 Tab，并优先把当前剪贴板内容放进去。

- 剪贴板是文字时，创建文本槽位。
- 剪贴板是图片时，创建图片槽位。
- 如果浏览器 / 环境拿不到真实剪贴板内容，则用兜底示例内容保证流程可见。

### 删除 Tab

点击 `−` 删除当前 Tab。

- 删除后会切到相邻 Tab。
- 至少保留一个槽位。

### 复制

点击复制按钮，只复制当前槽位内容。

- 文本槽位复制文本。
- 图片槽位复制图片。
- 复制按钮不弹原生提示，不自动全选文本。

### 隐藏与唤醒

点击隐藏按钮或按快捷键会收起浮窗。

- 收起时向右侧滑出。
- 唤醒时从右侧边缘磁吸式拉出。
- 右侧唤醒条只在完全隐藏后可见，点击后立即消失。

### 内容切换

切换 Tab 时，内容区会有很轻的换槽反馈，但外框高度不变。

## 内容类型

### 文本槽位

- 一个固定高度的透明文本框。
- 支持直接编辑。
- 内容过长时在槽位内部滚动。
- 可使用等宽字体显示代码或命令感内容。

### 图片槽位

- 一个轻薄照片贴片。
- 几乎无框，图片本身是主体。
- 图片在固定槽位内铺满显示，可轻微裁切。
- 图片信息只保留极简类型与尺寸。

## 视觉方向

- 透明、轻薄、克制。
- 白色半透明玻璃感，但不做亚克力夹片那种重装饰。
- 文本和图片属于两种不同物件，但都保持轻。
- Tab 按钮尺寸固定，切换时只变颜色，不变尺寸。
- 主要动效是磁吸展开和轻微内容换槽，不做炫技式多段动画。

## 一句话规格

GlazePad 是一个 Windows 桌面上的透明短浮窗，每个 Tab 是一个独立暂存槽，点击加号会根据当前剪贴板自动新增文本或图片槽位，之后通过复制按钮快速复用，必要时用磁吸式隐藏 / 唤醒从桌面退场和回到桌面。
