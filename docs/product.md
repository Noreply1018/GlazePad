# GlazePad 产品说明

## 产品定位

GlazePad 是一个运行在 Windows 上的轻量侧边栏工具，用来保存短时间内会反复使用的文字和图片。

它不是自动记录所有内容的剪贴板历史工具，也不是完整笔记软件，而是一个主动收纳、快速复用的临时工作区。

## 核心价值

- 替代把内容临时放在微信输入框、文件传输助手、聊天窗口里的工作习惯。
- 让用户可以把已经复制好的文字或图片快速收进一个侧边栏。
- 后续需要复用时，通过明确的复制按钮快速放回系统剪贴板。
- 保持轻量、克制、低干扰，不占用太多屏幕注意力。

## 目标用户

第一目标用户是开发者本人，使用场景以 Windows 桌面办公、开发、资料整理、沟通回复为主。

典型场景包括：

- 临时保存几段即将反复发送的话术。
- 保存几段代码片段或命令，短时间内多次复制。
- 保存截图、设计参考、小素材，后续反复粘贴到其他软件。
- 临时记录订单号、链接、备注等短文本。

## 产品形态

- Windows 右侧侧边栏。
- 默认通过全局快捷键唤醒或隐藏。
- 第一版使用置顶侧边栏窗口，不优先实现 AppBar 挤压屏幕。
- 支持手动调整侧边栏宽度。
- 视觉风格采用高级浅色、简洁白框、轻量半透明质感，贴近 Windows 11 的克制界面风格。

## 第一版 MVP

第一版只实现核心链路：

1. 通过快捷键唤醒或隐藏右侧侧边栏。
2. 在侧边栏中管理多个 Tab。
3. 点击加号读取当前系统剪贴板。
4. 剪贴板为文字时，生成文本卡片。
5. 剪贴板为图片时，生成图片卡片。
6. 点击卡片上的复制按钮，将对应内容写回系统剪贴板。
7. 点击文本内容进入编辑状态。
8. 支持删除卡片。
9. 支持清空当前 Tab。
10. 本地半永久保存数据，软件关闭或电脑重启后仍然保留。

## 明确不做的第一版功能

以下功能暂时不进入 MVP：

- 自动记录所有剪贴板历史。
- 云同步。
- 富文本编辑。
- Markdown 渲染。
- 代码高亮。
- 搜索。
- 敏感内容锁定。
- AppBar 挤压屏幕。
- 鼠标贴边自动唤醒。
- 拖拽内容到其他软件。
- 图片标注。

这些功能不是永久排除，而是在核心体验稳定后再根据实际使用情况决定。

## 一句话规格

GlazePad 第一版是一个 Windows 右侧常驻轻量侧边栏，通过快捷键唤醒，点击加号主动收纳当前剪贴板里的文字或图片，并以 Tab 加卡片的形式半永久保存，后续通过卡片复制按钮快速复用。
